Angular Pipes

Mangalino, Gianne Aisle B. | WD-303

Date Pipes

Formats a date object into a readable string.

Standard: {{presentDate | date }}

Short: {{presentDate | date:'shortDate' }}

Full: {{presentDate | date:'fullDate' }}

Async Pipe

Unwraps values from asynchronous primitives like Observables.

Chained: {{presentDate | date:'fullDate' | uppercase}}

Clock: {{ time$ | async | date:'mediumTime' }}

Currency Pipe

Transforms numbers into formatted currency strings.

Default: {{ price | currency:'':true }}

INR: {{ price | currency:'INR' }}

Slice Pipe

Returns a subset of an array based on indices.

Range (1:4): {{ Fruits | slice:1:4 }}

Negative (-2): {{ Fruits | slice:-2 }}

Decimal Pipe

Formats numbers with specific decimal places.

Format 3.1: {{ decimalNum1 | number:'3.1' }}

Format 2.1-4: {{ decimalNum1 | number:'2.1-4' }}

TitleCase Pipe

Capitalizes the first letter of each word.

Result: {{ testTitle | titlecase }}

KeyValue Pipe

Transforms an object into an array of key-value pairs.

{{ item.key | titlecase }}: {{ item.value }}

Json Pipe (Debug)

Converts objects into JSON strings for debugging. 
{{ objectData | json }}
